# 百炼接口文档

本文档整理本项目接入阿里云百炼时需要用到的接口规范。当前阶段优先接入 Qwen-ASR Flash，用于把前端上传的音频识别成文本。

## 官方依据

- Qwen-ASR API 参考：<https://help.aliyun.com/zh/model-studio/qwen-asr-api-reference>
- 录音文件识别：<https://help.aliyun.com/zh/model-studio/qwen-speech-recognition>
- 英文 Qwen-ASR API 参考：<https://www.alibabacloud.com/help/en/model-studio/qwen-asr-api-reference>

## 模型选择

短音频识别使用：

```text
qwen3-asr-flash
```

官方文档说明 Qwen3-ASR-Flash 适合短音频实时识别，音频大小限制为 10 MB。长音频或批量转写应改用异步文件转写模型 `qwen3-asr-flash-filetrans`。

## HTTP 接口

本项目不使用 SDK，直接通过 HTTP 请求调用 OpenAI 兼容接口。

北京地域：

```text
POST https://dashscope.aliyuncs.com/compatible-mode/v1/chat/completions
```

新加坡地域：

```text
POST https://dashscope-intl.aliyuncs.com/compatible-mode/v1/chat/completions
```

美国地域：

```text
POST https://dashscope-us.aliyuncs.com/compatible-mode/v1/chat/completions
```

## 请求头

```http
Authorization: Bearer <百炼 API Key>
Content-Type: application/json
```

API Key 必须从后端 `.env` 配置读取，不从系统环境变量读取。

## Base64 音频输入

官方支持 Base64 Data URL，格式为：

```text
data:<mediatype>;base64,<data>
```

示例：

```text
data:audio/webm;base64,<base64音频内容>
data:audio/wav;base64,<base64音频内容>
data:audio/mpeg;base64,<base64音频内容>
```

注意：Base64 会增大体积，需要控制编码后的音频仍符合模型输入限制。

## 请求体

```json
{
  "model": "qwen3-asr-flash",
  "messages": [
    {
      "role": "user",
      "content": [
        {
          "type": "input_audio",
          "input_audio": {
            "data": "data:audio/webm;base64,<base64音频内容>"
          }
        }
      ]
    }
  ],
  "stream": false,
  "asr_options": {
    "enable_itn": false
  }
}
```

可选参数：

- `asr_options.language`：已知语种时可指定，例如 `zh`。
- `asr_options.enable_itn`：是否开启逆文本规范化。本项目先从 `.env` 读取，默认 `false`。

## 响应解析

非流式响应主要读取：

```text
choices[0].message.content
```

如果后续启用流式输出，需要在请求头中指定：

```http
X-DashScope-SSE: enable
```

并拼接每个 chunk 中的 `choices[0].delta.content`。

## 本项目后端配置

配置文件：

```text
backend/.env
```

当前使用的配置项：

```env
BAILIAN_API_KEY=
BAILIAN_ASR_ENDPOINT=https://dashscope.aliyuncs.com/compatible-mode/v1/chat/completions
BAILIAN_ASR_MODEL=qwen3-asr-flash
BAILIAN_ASR_TIMEOUT_SECONDS=60
BAILIAN_ASR_ENABLE_ITN=false
BAILIAN_ASR_LANGUAGE=
BAILIAN_ASR_SYSTEM_PROMPT=
BAILIAN_ASR_MAX_AUDIO_BASE64_MB=10
```

## 安全与网络约束

- 不使用 SDK，只使用 HTTP 请求。
- 初始化百炼 HTTP 客户端前，清除进程环境变量中的代理变量，例如 `HTTP_PROXY`、`HTTPS_PROXY`、`ALL_PROXY`、`NO_PROXY` 及大小写变体。
- HTTP 客户端也显式禁用代理，避免系统代理影响百炼调用。
- 日志和返回体中不要输出完整 API Key。
- 原始音频和 ASR 输出都保存到 `Storage` 目录，便于排查完整链路。
